FINGER GUN

OVERVIEW

Finger gun (solo project) is a machine learning game in which the player uses their hands to aim at the enemies that are approaching the player. The game uses three different software: Unity, Processing and Wekinator. Wekinator communicates with Processing that handles the features of a hand skeleton (input) and Unity which handles the game (output) and grabs the hand mechanic features from Processing. The languages that this project uses were Java, JavaScript and C#.

DESIGN & IMPLIMENTATION

The enemy was created using 5 sphere objects, one for the body, one as the ring, one for the eyeball and two for the eye shape; the textures for the game done on Photoshop and exported as a png to the 'Texture' file.
