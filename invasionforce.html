INVASION ALPHA FORCE

❮ ❯

OVERVIEW

Invasion Alpha Force (team project) is a top down 2D turn-based RPG in which you play as aliens who are invading Earth to gather resources of militant bases. The three aliens in the force are a melee, long range and a healer. The game was created using Gamemaker (GML) and created the sprite sheets using Asprite.

DESIGN & IMPLIMENTATION

DESIGN

The characters were designed and animated using Asprite in which it allowed to layer and tag functionalities for each characters movements, death and attacks.

The level design was created using Gamemakers in built toolkit and added onto the level builder to create the first level for the game.

IMPLIMENTATION

The implementation for this project was using GameMaker’s inbuilt sprite animation and its own language, where each script dealt with the character’s movements, attack system, collision, health and turn based system.
